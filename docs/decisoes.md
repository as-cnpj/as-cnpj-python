# Decisões

Idiomas: **Português (Brasil)** | [English](decisoes.en.md) | [Español](decisoes.es.md) | [Français](decisoes.fr.md)

## 1. Python puro

- zero dependência de runtime;
- fácil auditoria;
- port direto do contrato do ecossistema.

## 2. API idiomática em snake_case

- mantém equivalência sem forçar convenções de outra linguagem;
- reduz ruído em bases Python.

## 3. Vetores compartilhados do hub

- Python não vira dono exclusivo da verdade;
- os demais ports continuam comparáveis.
